# 📢 Marketing Guide

## 🎯 คู่มือการตลาดเกม Zombie Quiz RPG

### Marketing Strategy Overview

#### 4Ps of Marketing

**1. Product (ผลิตภัณฑ์)**
- เกม RPG ควิซเตรียมสอบราชการ
- 2 ภาษา (ไทย/อังกฤษ)
- 20 ด่าน, 8 ตัวละคร
- Free-to-play

**2. Price (ราคา)**
- ฟรีในการเล่น
- In-app purchases: $0.99 - $39.99
- Subscription: $4.99 - $19.99/month

**3. Place (ช่องทางจัดจำหน่าย)**
- Apple App Store
- Google Play Store
- Website (สำหรับข้อมูล)

**4. Promotion (การส่งเสริมการตลาด)**
- Social media marketing
- Influencer marketing
- Content marketing
- Paid advertising
- Community building
- PR & press releases

## 🎯 Target Audience

### Primary Audience
**นักเรียน/นักศึกษาเตรียมสอบ**
- อายุ: 18-25 ปี
- เพศ: ชาย/หญิง
- สถานที่: ทั่วประเทศ
- ความสนใจ: สอบราชการ, เรียน, เกมมือถือ
- Pain Point: เบื่อการอ่านหนังสือ, ต้องการวิธีเรียนสนุกๆ

### Secondary Audience
**คนทำงานเตรียมสอบ**
- อายุ: 25-35 ปี
- เพศ: ชาย/หญิง
- สถานที่: เมืองใหญ่
- ความสนใจ: เลื่อนตำแหน่ง, สอบราชการ
- Pain Point: ไม่มีเวลาเรียน, ต้องการเรียนแบบ mobile

### Tertiary Audience
**ผู้ปกครอง**
- อายุ: 40-55 ปี
- เพศ: ชาย/หญิง
- ความสนใจ: การศึกษาของลูก
- Pain Point: ลูกไม่ชอบเรียน
- Use Case: ซื้อให้ลูกเล่น

## 📣 Paid Advertising

### Facebook Ads

#### Campaign Objectives
1. **App Installs** - เน้นการติดตั้งแอพ
2. **Engagement** - เน้นการมีส่วนร่วม
3. **Conversions** - เน้นการซื้อของในแอพ

#### Targeting
```
Location: Thailand
Age: 18-35
Interests:
- การศึกษา
- สอบราชการ
- กพ.
- เกมมือถือ
- เกม RPG
- เกมควิซ
- การเรียนรู้

Behaviors:
- Mobile device users
- App purchasers
- Engaged shoppers
```

#### Ad Formats

**1. Image Ads**
- ขนาด: 1080x1080px (Feed), 1080x1920px (Stories)
- ข้อความสั้น กระชับ
- ภาพสีสันสดใส

**Ad Copy ตัวอย่าง:**
```
Headline: ฝึกทำข้อสอบ กพ. อย่างสนุก!
Primary Text: 
🎮 Zombie Quiz RPG เกมควิซเตรียมสอบราชการ
📚 ข้อสอบจริง 6 วิชา
🏆 เล่นฟรี! ดาวน์โหลดเลย

Link: [App Store Link]
```

**2. Video Ads**
- ความยาว: 15-30 วินาที
- Square (1080x1080) หรือ Vertical (1080x1920)
- Hook ใน 3 วินาทีแรก
- Text overlay (คนดูส่วนใหญ่ปิดเสียง)

**3. Carousel Ads**
- 3-5 cards
- แต่ละ card แสดงฟีเจอร์ต่างกัน
- Card สุดท้าย = Call to Action

**4. Playable Ads**
- ให้ลองเล่น demo สั้นๆ
- 15-30 วินาที
- Install button ตลอดเวลา

#### Budget & Bidding
```
Daily Budget: 1,000 - 10,000 THB
Bid Strategy: Lowest Cost
Cost Cap: 50 THB per install

Expected Results:
- CPI (Cost Per Install): 30-60 THB
- CTR (Click-Through Rate): 1-3%
- Conversion Rate: 20-30%
```

### Influencer Marketing

#### Types of Influencers

**Mega Influencers (1M+ followers)**
- Reach: 1M+
- Cost: 100,000 - 500,000+ THB
- Best for: Brand awareness

**Macro Influencers (100K-1M followers)**
- Reach: 100K-1M
- Cost: 30,000 - 100,000 THB
- Best for: Reach + engagement

**Micro Influencers (10K-100K followers)**
- Reach: 10K-100K
- Cost: 5,000 - 30,000 THB
- Best for: Engagement + conversion

**Nano Influencers (1K-10K followers)**
- Reach: 1K-10K
- Cost: 500 - 5,000 THB หรือ product exchange
- Best for: Authenticity + niche

**1. Sponsored Video**
- Influencer สร้างวิดีโอเกี่ยวกับเกม
- ความยาว: 3-10 นาที
- รวม gameplay จริง

**Cost:** 5,000 - 50,000 THB

**Commission:** 10-30% of revenue

## 📊 Marketing Metrics

### Key Metrics to Track

#### Acquisition
- **Installs**: Total new installs
- **CPI**: Cost Per Install
- **Organic vs Paid**: % from each channel
- **Source**: Which channel brings most users

#### Engagement
- **DAU/MAU**: Active users
- **Session Length**: Time spent
- **Retention**: D1, D7, D30
- **Social Shares**: Viral coefficient

#### Monetization
- **Revenue**: Total revenue
- **ARPU**: Average Revenue Per User
- **Conversion Rate**: % paying users
- **LTV**: Lifetime Value

#### Marketing Efficiency
- **ROI**: Return on Investment
- **ROAS**: Return on Ad Spend
- **LTV/CAC**: Lifetime Value / Customer Acquisition Cost
- **Payback Period**: Time to recover CAC

### Tracking Setup

#### UTM Parameters

- **Facebook**: Facebook Pixel, SDK
- **Google**: Google Analytics, Firebase
- **Attribution Platform**: AppsFlyer, Adjust, Branch

### Reporting

- Installs by source

- All daily metrics
