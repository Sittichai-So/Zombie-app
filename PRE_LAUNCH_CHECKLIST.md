# ✅ Pre-Launch Checklist

## 🎯 สิ่งที่ต้องทำก่อนปล่อยเกม

### 📝 Content & Data

#### ข้อสอบ

- [ ] ตรวจสอบความถูกต้องของคำตอบ
- [ ] เพิ่มคำอธิบายที่ชัดเจน
- [ ] จัดหมวดหมู่ให้ชัดเจน
- [ ] เพิ่มระดับความยาก (Easy, Medium, Hard)
- [ ] ตรวจสอบภาษาไทยถูกต้อง
- [ ] ตรวจสอบภาษาอังกฤษถูกต้อง

#### ตัวละคร
- [ ] สร้าง/ซื้อ sprites จริง (แทน emoji)

## 📊 Success Metrics

### KPIs to Track
- **Downloads**: จำนวนดาวน์โหลด
- **DAU/MAU**: Daily/Monthly Active Users
- **Retention**: Day 1, 7, 30 retention
- **Session Length**: ระยะเวลาเล่นเฉลี่ย
- **Revenue**: รายได้ต่อวัน/เดือน
- **ARPU**: Average Revenue Per User
- **Rating**: คะแนนเฉลี่ยใน store
- **Crash Rate**: อัตราการ crash

### Goals (ตัวอย่าง)
- เดือน 1: 1,000 downloads, 4.0+ rating
- เดือน 3: 10,000 downloads, 4.5+ rating
- เดือน 6: 50,000 downloads, profitable

## 🎯 Priority Matrix

### High Priority (ต้องทำ)
1. เพิ่มข้อสอบให้เพียงพอ
2. แก้ไข bug ทั้งหมด
3. สร้าง graphics จริง
4. เพิ่มเสียงประกอบ
5. ทดสอบอย่างละเอียด

### Medium Priority (ควรทำ)
1. เพิ่ม achievements
2. สร้าง tutorial
3. เพิ่มอนิเมชั่น
4. ปรับปรุง UI
5. ตั้งค่า analytics

### Low Priority (ทำทีหลัง)
1. ระบบ social
2. Leaderboard online
3. PvP mode
4. Special events
5. Merchandise
